#После создания репозитория на удаленном сервере, синхронизации его локальным репозиторием и, после того, как "запушили", увидели, что файл из локального репозитория появился в удаленном. Теперь вносим изменения в файл на локальном репозитории, сохраняем, проверяем наличие изменений в файте в командной строке Git CMD командной:
  C:\Users\andre\MyProjects\github>git status
(видим, что есть изменения - красными буквами)

#Далее команда:
  C:\Users\andre\MyProjects\github>git commit

#Добавляем эти изменения:
  C:\Users\andre\MyProjects\github>git add my.html.txt

#Открываем окно vim и прописываем в верхнюю строку: Add Hello, закрываем окно командами Esc, Shitf+Z, Shift+z
    C:\Users\andre\MyProjects\github>git commit

#Пушим:
  C:\Users\andre\MyProjects\github>git push

#Заходим в профиль GitHub, обновляем репозиторий и видим изменения внутри самого файла.
